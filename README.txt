박주환 포트폴리오 클린 버전

구성
- index.html
- projects/har.html
- projects/weather-maru-frost.html
- projects/weather-index.html

사용 방법
1. ZIP 파일을 압축 해제합니다.
2. 압축 해제된 폴더의 index.html을 더블클릭합니다.
3. 메인 페이지의 프로젝트 상세보기를 누르면 각 프로젝트 페이지로 이동합니다.

주의
- ZIP 내부에서 바로 index.html을 열지 말고 반드시 압축 해제 후 열어야 합니다.
- 프로젝트 상세 페이지는 스크롤이 아니라 좌우 버튼, 키보드 ←/→, 마우스 휠로 페이지가 넘어갑니다.

반영 사항
- 기술스택 메뉴 삭제
- 좌측 하단 개인정보 박스 삭제
- 우상단 메일 보내기 삭제
- 섹션 설명 문구 삭제
- 대시보드 하단 최근 프로젝트/핵심 역량 삭제
- 주소는 '경기도 고양시'까지만 표시
- 학교명은 '한국항공대학교'로 표시
- 라이트모드 기본값
