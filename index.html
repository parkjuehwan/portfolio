인간 행동 인식(HAR)

2024.09 – 2025.07 · 졸업 논문 PDF ↗

저해상도 적외선(IR) 센서 데이터를 활용해 사람의 자세와 동작을 인식하는 Human Activity Recognition 프로젝트입니다. 단일 프레임 기반 CNN과 연속 시퀀스 기반 CNN-LSTM을 비교해 해상도 저하가 인식 성능에 미치는 영향을 분석했습니다.

PROBLEM

기존 RGB·고해상도 영상의 한계

  • 고해상도 RGB 영상은 인식 정확도는 높지만 개인 식별 가능성이 큽니다.
  • 병실, 화장실, 독거노인 주거공간처럼 사생활 보호가 중요한 공간에 적용하기 어렵습니다.
  • 조도가 낮은 환경에서는 영상 기반 인식 성능이 저하될 수 있습니다.

저해상도 IR 센서의 가능성

  • 비가시광 열 정보를 사용해 조명 환경의 영향을 줄일 수 있습니다.
  • 극저해상도에서는 인체 세부 식별이 어려워 사생활 보호에 유리합니다.
  • 다만 공간 정보가 크게 줄어들어 인식 정확도 저하 문제가 발생합니다.

PREPROCESSING

datetime 인덱스 생성

연도, 월, 일, 시 컬럼을 하나의 datetime 형식으로 통합했습니다. 24시는 0시로 변환하고 생성한 datetime을 인덱스로 설정했습니다.

연도 + 월 + 일 + 시 → datetime index

Forward Fill 결측치 처리

기온, 상대습도, 풍속은 시간적으로 완만하게 변화하는 기상 변수라는 점을 고려해 바로 이전 시점의 값을 채우는 ffill 방식을 적용했습니다.

missing weather value → previous time value
FEATURES

주기성 변환

시와 월은 순환 변수이므로 sin/cos 변환을 적용해 시간의 순환적 특성을 모델이 학습하도록 했습니다.

파생 변수

주말 여부, 냉방도일, 난방도일, 불쾌지수를 생성해 생활 패턴과 기상 조건을 반영했습니다.

Lag / Rolling

24시간 전, 168시간 전 전력부하와 기온, 최근 24시간 평균 기온을 추가했습니다.

불쾌지수 = 1.8 × 기온 − 0.55 × (1 − 상대습도/100) × (1.8 × 기온 − 26) + 32

특성 공학 후 모델 입력 변수는 기존 14개에서 28개로 확장되었습니다.
